# 🧮 Math Projects

A collection of mathematical foundations behind Machine Learning and Artificial Intelligence implemented from scratch using Python and NumPy.

The purpose of this repository is to understand the mathematics and intuition behind modern AI algorithms instead of treating them as black boxes.

---

## 📚 Projects

### 🟢 Cosine Similarity Search

Build a mini semantic search engine from scratch.

**Topics Covered**

- Text Preprocessing
- Vocabulary Building
- Bag of Words
- Count Vectorizer
- TF-IDF (Concept)
- Cosine Similarity
- Semantic Search
- Vector Space Representation

**Tech Stack**

- Python
- NumPy
- Matplotlib
