# 🧮 Math Projects for AI & Machine Learning

This repository contains mathematical foundations behind modern Machine Learning and Deep Learning algorithms, implemented **from scratch** using Python and NumPy.

The goal is not just to use machine learning libraries, but to understand **how the algorithms actually work internally** by implementing them step by step.

---

## 📌 Tech Stack

- Python
- NumPy
- Matplotlib

---

# Projects

## 🟢 1. Cosine Similarity Search

A complete implementation of semantic search from scratch.

### Features

- Implement cosine similarity using pure NumPy
- Build a mini semantic search engine
- Convert text into vectors
- Rank sentences based on similarity
- Visualize vector space using Matplotlib

### Concepts Covered

- Vector Mathematics
- Dot Product
- Magnitude
- Cosine Similarity
- Text Vectorization
- Semantic Search
- NLP Fundamentals
